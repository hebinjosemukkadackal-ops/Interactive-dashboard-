# Superstore Sales Dashboard - README

## Overview

The Superstore Sales Dashboard is an interactive data visualization tool designed to provide actionable insights into sales performance, customer behavior, and regional trends. This dashboard is based on a dataset containing over 5,900 rows of retail sales data from a superstore in the United States. It leverages charts, maps, and filters to give users a comprehensive view of the business metrics.

## Dataset Summary

* **Total Records**: 5,901
* **Columns**: 23
* **Time Range**: Multiple years (includes Order Dates and Ship Dates)
* **Primary Metrics**: Sales, Profit, Units Sold, Returns, Ship Mode, Category, Sub-Category

## Key Columns

| Column Name    | Description                              |
| -------------- | ---------------------------------------- |
| Order ID       | Unique identifier for each transaction   |
| Order Date     | Date of the order                        |
| Ship Date      | Date the product was shipped             |
| Customer Name  | Name of the customer                     |
| Category       | Main product category                    |
| Sub-Category   | Detailed product type                    |
| Sales          | Revenue generated                        |
| Profit         | Profit earned from the sale              |
| Quantity       | Number of units sold                     |
| Discount       | Applied discount                         |
| Region         | Sales region                             |
| State          | US state where the order was shipped     |
| Ship Mode      | Delivery method                          |
| Payment Method | Mode of transaction (Online, Cards, COD) |
| Return Status  | Whether the unit was returned            |

## Dashboard Features

* **Revenue, Profit, Units Sold, and Returns Summary**
* **Sales Breakdown by Category and Sub-Category**
* **Monthly Year-over-Year Profit and Sales Trends**
* **Sales by Ship Mode**
* **Geographical Sales Distribution by State**
* **Payment Methods Pie Chart**
* **Interactive Filters**: Allows filtering by customer name and other dimensions

## Tools Used

* Microsoft Excel (for data preparation)
* Power BI (for visualization and reporting)
* Python Libraries (Optional): pandas, matplotlib, seaborn (for automated reporting)

## Instructions for Use

1. Open the dashboard file in your preferred visualization tool.
2. Use the filters to drill down into specific customer data or product categories.
3. Hover over the charts for detailed information.
4. Export or print insights as needed for reporting.

## Summary of Insights

* **Top Performing Categories**: Office Supplies and Phones generated high revenue.
* **Seasonality Observed**: Q4 months (October to December) saw peak sales and profit.
* **Preferred Ship Mode**: Standard Class dominates shipping methods.
* **Returns Insight**: Approximately 5.9K units were returned, impacting net profits.
* **Payment Trends**: COD is the most used method, followed by Online and Cards.
* **State Performance**: Certain states like California and New York showed high sales volumes.

## Recommendations

* Optimize inventory based on top-performing sub-categories.
* Focus marketing efforts during peak sales months.
* Streamline operations for faster ship modes.
* Investigate reasons behind high return rates.
